# CRYPTOCURRENCY
This project aims to develop a sophisticated risk analysis system which is capable of providing analysis for cryptocurrency transactions to determine if there is any potential risk involved. This project focuses on analyzing the risks associated with cryptocurrency and developing a risk management framework using Hierarchical Risk Parity (HRP) and supervised machine learning. It aims to assess the intrinsic risks of cryptocurrency, particularly regarding the likelihood of occurrence and financial impact.
